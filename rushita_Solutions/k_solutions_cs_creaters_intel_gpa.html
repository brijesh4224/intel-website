What's New: Release 2024.2

  • Enhanced UX/UI in Graphics Monitor to improve usability and product onboarding.
  • Enhanced shader analysis: Added compiler flags and ISA assembly headers. Enabled the ISA code update for an experiment with shader code modification.
  • Improved visualization for graphics pipeline states and added error messaging for an experiment with pipeline state modification.
  • Added new accessibility settings in Graphics Frame Analyzer, including three options for the Color Blindness scheme and the Use Patterns mode.
  • Added functionality to the Intel GPA framework:
    • Enhanced single-threaded and multithreaded playback with up to a 50% performance improvement for Direct3D* 12.
    • Enabled support for the majority of interfaces defined in the latest Agility SDK.
    • Significantly increased stability with improvements and bug fixes.
