Date Code Decoder

What do the alphanumerics in the device date code represent?

Description

The device date code for Intel® FPGAs is listed under the device part number on the package. The date code format is same for all FPGA and CPLD devices.

It contains 11 alphanumerics as shown: A XβZααYYWWT

Date Code Decoder

  • A = Fab Process Identifier
  • X = Final Test Location
  • β = Base Die Identifier
  • Z = Die Revision Level
  • αα = Fab Process Code
  • YY = Year
  • WW = Work Week
  • T = Intel FPGA Internal Identifier
