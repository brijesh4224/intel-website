Get Started

The AI PC Development Kit from Intel is a powerful AI PC with a prevalidated, comprehensive AI software stack (SDK) that includes tools and code samples, making it ideal for independent and enterprise software developers to jump-start their AI development.

 

Setup
  1. Install AI PC software on your Dev Kit by opening a command terminal with Administrator privileges and running the installer.exe located in IntelAIPCDevKit folder on the desktop.

    If you don't have an AI PC Dev Kit:

    •  See purchase options below.
    •  Access the free AI PC Cloud
    •  Go to the open source page to build your own installer to use on any compatible AI PC.

  2. After installation is complete, a Jupyter Lab session is started by default.

  3. Choose a deployment option below.

 

The AI PC Development Kit Includes:

  • OpenVINO™ toolkit: A powerful AI inferencing framework.
  • OpenVINO™ notebook: A Jupyter* Notebook contains over 100 samples that demonstrate the OpenVINO toolkit development lifecycle on real use cases.
  • OpenVINO toolkit samples: Sample applications that use the toolkit and show real-world use cases.
  • Python*: A programming language.
  • Git: A source-code tool for working with software code repositories.
  • Vim: A powerful text editor.
  • Microsoft Visual Studio* Code: A powerful text editor with many community-developed enhancements.
  • CMake*: A software build tool.
  • OpenCV: A compute vision library for high performance.
  • Intel® Driver & Support Assistant: Updates system-level software on the AI PC.
  • WebNN links: Provide access to getting started resources and samples.

 

For AI PC using OpenVINOTM Toolkit

  1. After installation is complete, a Jupyter Lab session is started by default. You can also launch this manually:

    1. Start a command prompt
    2. Navigate to the install directory: cd c:\intel\aipcdevkit
    3. Activate the environment: aipc-venv\scripts\activate.bat
    4. Navigate to the OpenVINO notebooks: cd openvino_notebooks
    5. Launch the Jupyter Notebook server: jupyter lab notebooks

  2. Copy and paste one of the URLs displayed in the cmd window output for the Jupyter Lab server to your browser.

  3. Once the browser is connected to the jupyter Lab session and the file system displays, open the notebooks folder then the project hello_npu folder.

  4. Double click on hello_npu.ipynb and follow the instructions to execute the Jupyter notebook example. This will help verify that your installation succeeded and detect available devices, including the NPU.

 

For AI PC Using ONNX* with OpenVINO Toolkit
  1. Open File Explorer and navigate to the following: C:\Intel\aipcdevkit\openvino_notebooks\openvino_build_deploy\notebooks\onnxruntime_lcm\
  2. Open the README.md.
  3. Right-click in the File Explorer to open a command terminal in the same directory as the README.md.
  4. Follow the Installation Instructions to setup the environment and install dependencies.
  5. Launch the Jupyter Notebook server: jupyter lab notebooks

Double click on the .ipynb file and follow the instructions to execute the Jupyter notebook example.

 

For AI PC for the Web

WebNN requires a compatible browser to run, and Windows* 11 v21H2 (Direct Machine Learning [DirectML] 1.6.0) or higher. Try the latest Google Chrome* Canary or Microsoft Edge* Canary, which requires enabling WebNN functionality in the settings.

  1. Download the latest Google Chrome Canary or Microsoft Edge Canary browser.
  2. To enable WebNN, in your browser address bar, enter chrome://flags, and then press Enter. An experiments page opens.
  3. In the Search flags box, enter webnn. Enables WebNN API appears.
  4. In the drop-down menu, select Enabled.
  5. Relaunch your browser.
  6. Go to the WebNN Github repository for additional setup instructions and samples.

Workshop: How to Use WebNN for AI PC Application Development Learn how the WebNN API is standardizing interactions with CPUs, GPUs, and NPUs to gain optimal performance and predictable behavior.

Develop in the Cloud with AI PC Cloud

The AI PC Cloud offers developers free 24x7 access to an AI PC through Intel® Tiber™ AI Cloud, empowering developers with a cost-effective, resource-rich environment to explore and build AI solutions.

  • Free Access to Cutting-Edge Hardware: Developers can use the newest Intel Core Ultra processors to test, develop, and refine AI applications on advanced technology at no cost.
  • Optimized Software Tools: The platform includes a tailored software stack from Intel, featuring tools and libraries designed to enhance AI development. These optimizations streamline workflows, improve performance, and reduce the time spent on setup and configuration.
  • Enterprise-Grade Security: Protect your data and AI models with robust security features from Intel, including encryption, access controls, and compliance with global standards like General Data Protection Regulation (GDPR) and International Organization for Standardization (ISO).
  • Hands-On Learning Opportunities: With access to high-performance AI PCs, developers can experiment and gain practical experience with the latest AI technologies. This hands-on approach is ideal for skill-building and staying current with industry advancements.

The AI PC Cloud provides the resources and infrastructure you need to bring your AI ideas to life.

  1. Sign in or create your free account.
  2. After you sign in, if AI PC hardware options do not display, in the left navigation, select Preview > Preview Catalog to display the AI PC options.
  3. Select either of the valid AI PC options, and then follow the prompts to request and launch the instance.
  4. Follow the OpenVINO Toolkit get started instructions.
