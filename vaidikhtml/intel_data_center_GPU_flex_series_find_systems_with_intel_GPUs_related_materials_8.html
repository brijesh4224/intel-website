Executive Overview

Intel® Optane™ Persistent Memory addresses issues of increased I/O overheads and decreased computing performance as data spills to traditional HDD during scientific computing of Mars, a unified distributed computing framework based on tensors. Mars was launched by the R&D team of MaxCompute, a big data computing platform of Alibaba.

Tests prove that, with the same TCO, Intel Optane PMem can effectively improve scientific computing performance of Mars and make scientific computing more efficient.
