Agilex™ FPGA Development Kits

Intel provides a range of development kits based on Agilex™ FPGAs, which can accelerate the design process. These FPGA and SoC FPGA designs are available in tabletop and PCIe form factors to cater to general-purpose broad market requirements. They support various connector interfaces for transceivers and FPGA workloads. These kits also come equipped with on-board memory and memory upgrade with HPS enabled hardware options.

Browse the Development Kits

F-Series Dev Kit (P-Tile and E-Tile)

Use this kit to develop and test PCIe 4.0 design using the PCI-SIG compliant development board. You can also evaluate SoC features and performance using the Hard Processor System (HPS).
F-Series Transceiver-SoC Dev Kit (P-Tile & E-Tile)

Use this kit to develop and evaluate E-Tile transceivers up to 58 Gbps and Hard Processor System (HPS). Suitable for Networking and Communication workloads and solutions to utilize Advanced DSP Blocks and high-performance crypto blocks.
F-Series Dev Kit (2x F-Tile)

Use this kit as a general-purpose evaluation board in PCIe form factor with Hard Processor System (HPS) features to develop your solution.

I-Series Dev Kit (2x R-Tile & 1x F-Tile)

Use this kit to develop and test PCIe 5.0, CXL v1.1/2.0, high-speed Ethernet up to 400G designs using applicable IP from Intel or third parties.
I-Series Transceiver-SoC Dev Kit (4x F-Tile)

Use this kit to evaluate F-Tile FGT and FHT transceiver performance and develop your solutions for flexible protocol support in Video and Vision, JESD204C, Serial Lite IV, CPRI, and other major standards.
I-Series Transceiver Dev Kit (6x F-Tile)

Use this kit to evaluate F-Tile FGT and FHT transceiver capabilities and develop your solution targeted for general purpose embedded design as well as Wireline OTN, muxponder, multiplexer, crosspoint switch, Wireless designs with FMC+ connectors.

M-Series Development Kit - HBM2e Edition (3x F-Tile & 1x R-Tile)

Use this kit to evaluate M-series and start designs in memory-bandwidth driven and compute-intensive applications across cloud, network and edge with HPC, storage, wireline, military, radar, video & image processing, and test equipment segments.

E-Series 065B Dev Kit (Premium)

This kit comes with extensive interfaces, PCIe 4.0 connectivity, and expandable memory in a tabletop form factor ideal for embedded and edge applications and more.
E-Series 065B Dev Kit (Modular)

This kit comes with a pluggable modular board with a carrier board, allowing a mix/match of devices and connector types to serve a variety of applications. The main modular board comes with its own set of features and connectors allowing it to function standalone. The carrier board expands its capabilities making it suitable for a variety of embedded designs and edge applications in a PCIe form factor.

 

  • Fully integrated CXL IP solution with PCIe 5.0 x16 (32 GT/s) performance.
  • CXL 1.1 compliant IP that combines both hard and soft resources:
    • R-Tile Hard IP (PHY IP) reduces logic resourcing and simplifies timing closure.
    • Soft wrapper for added design flexibility and easy integration.
  • Validated with 4th and 5th Gen Intel® Xeon® Scalable Processors.
  • Compute Express Link Consortium compatible with CXL certification software (CXL_CV_app).
  • CXL protocol support: CXL.io, CXL.mem, CXL.cache.
  • Device type support: Type 1, 2, and 3.
  • CXL 1.1. IP enhanced with 2.0 features and functionality.
  • Flexible device-attached memory subsystem architecture:
    • Allows FPGA developer to mix and manage different memory types for maximum scalability and application fit.
