Overview

Video is a primary way that we interact with the world, whether via broadcasting and streaming, in-cloud gaming, or remote desktop solutions. In all cases, video codecs are the linchpin that allows all interactions to happen.

This webinar focuses on the Intel® Video Processing Library (Intel® VPL), which is a single, video-focused API for encoding, decoding, and video processing across CPUs, GPUs, FPGAs, and more.

Join technical consulting engineer Mark Liu to tour the highlights of Intel VPL and how it can help you unlock media features across a broad range of accelerators.

Topics covered:

  • Enhanced features of Intel VPL including new C++ and Python* APIs and High Efficiency Video Coding (HEVC) encode capabilities (10 bit, 4-2-2, 4-4-4)
  • An exploration of graphics hardware interoperability
  • How to identify and enable new hardware targets, including access to new and prerelease development systems in the Intel® Developer Cloud
  • Hyper Encode
