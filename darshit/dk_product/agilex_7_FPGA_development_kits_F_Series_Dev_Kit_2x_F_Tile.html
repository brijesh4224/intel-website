Agilex™ 7 FPGA F-Series Development Kit (2x F-Tile)

The Agilex™ 7 FPGA development kit provides a hardware development platform for evaluating the performance and features of the Agilex™ 7 FPGA F-Series with two F-Tiles. It is a general purpose evaluation board in PCIe form factor with HPS hardware features.

Learn more about  Agilex™ 7 FPGA F-Series devices ›

Use this Agilex™ 7 FPGA F-Series FPGA Development Kit to:

  • Develop and test PCIe 4.0 designs using the PCI-SIG-compliant development board.
  • Evaluate SoC features and performance using the Hard Processor System (HPS).
