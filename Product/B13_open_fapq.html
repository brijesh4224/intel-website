Open FPGA Stack (OFS) is the first complete hardware and software infrastructure that is fully open sourced and comprises composable hardware code and upstreamed kernel code to kernel.org to enable a collaborative community of FPGA developers.

All source code and technical documentation is open source in the OFS GitHub and enables development using the Intel® Stratix® 10 FPGA and Intel Agilex® FPGA.

Customize Unique FPGA-based Boards and Platforms

Open FPGA Stack is a scalable, open source hardware and software infrastructure delivered via git repositories that provides an efficient approach to customize your own unique acceleration platform solutions using an Intel, 3rd party, or custom board. OFS is being used by Intel and selected third-party platforms featuring our Intel® Stratix® 10 FPGA, Intel Agilex® FPGA, and future Intel® FPGA device families.

Hardware developers can try out OFS on the reference platforms before porting to their own boards. Application developers can use our 3rd Party Board Catalog of OFS-enabled boards to select an option and get started.
