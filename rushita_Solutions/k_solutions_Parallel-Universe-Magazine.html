Issue 57, July 2024

Feature: AI PC Brings Larger LLM Development to Your Desk

This issue showcases a series of articles covering a variety of AI topics.

Download Issue 57 to find:

  • How it's possible to do serious development more conveniently and securely on a local system instead of automatically defaulting to remote data center or cloud systems to develop large models.
  • How to accelerate low-bit LLM inference while taking advantage of the GPT*-generated unified format.
  • How to use the new SYCL* back end to run llama.cpp (a lightweight, high-performance LLM framework that is gaining popularity) on Intel® GPUs.

Download the PDF

Contents

AI PC Brings Larger LLM Development to Your Desk

Explore how small form-factor PCs based on Intel® Core® Ultra processors can deftly run the Llama 3 70B parameter model locally and at a lower cost than a workstation.

Enterprise AI Art Exhibition at Intel Vision 2024

Get the steps for running an open source Stable Diffusion* model on Intel® Gaudi® AI accelerators to create your own unique work of art.

Run LLMs on Intel® GPUs Using llama.cpp

The newly developed SYCL* in llama.cpp—a light, open source LLM framework—enables developers to deploy on the spectrum of Intel GPUs and GPUs from other vendors.

Accelerate Memory-Bandwidth-Bound Kernels Using the Intel® Data Streaming Accelerator (Intel® DSA)

Explore techniques that combine the strengths of Intel® DSA with CPUs to boost performance using fewer CPU cores. The article includes STREAM benchmarks.

Low-Bit Quantized Open LLM Leaderboard

Introducing the low-precision quantized open LLM leaderboard, a new tool for finding high-quality models that can be deployed efficiently on a given client.

Accelerating GGUF Models with Transformers

Learn how to convert a PyTorch* model to the GPT-Generated Unified Format (GGUF), a new binary file format that optimizes the efficiency of LLM storage and processing.

Accelerate Simulations and Backpropagation with Python* and C++ Analytics

Learn about the architectural advantages of AADC (Algorithmic Adjoint Differentiation Compiler) and how it enhances the performance of resource-intensive financial computations.
