Benefits

Priority Support is paid product support that covers all of the components included in the respective toolkits. It provides the following benefits for one year:

  • Direct and private interaction with Intel’s support engineers, including the ability to submit confidential support requests
  • Accelerated response time for technical questions and other product needs
  • Ability to influence product features and quality
  • Priority assistance for escalated defects
  • Free download access to all new product updates and continued access to older versions of the product
  • Access to a vast library of self-help documentation built from decades of experience in creating high-performance code
  • Additional services at reduced cost, including on-site or online training and consultation by Intel technical consulting engineers

Intel® Parallel Studio XE and Intel® System Studio Users

If you have active Priority Support for either of these products, you qualify for a free transition to Priority Support for the corresponding Intel toolkit:

  • Intel® Parallel Studio XE transitions to the Intel® oneAPI Base & HPC Toolkit
  • Intel® System Studio transitions to the Intel® oneAPI Base & HPC Toolkit

Intel toolkits are the next generation of standards-based Intel software development tools used to build and deploy applications across diverse architectures. Transitioning to these toolkits provides you with the latest optimizations and fixes for the familiar tools you have been using, and delivers additional capabilities for software development that targets Intel CPUs, GPUs, and FPGAs.
