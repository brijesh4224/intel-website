Free IP Base Suite Licenses

To help shorten your design time, Intel provides full production licenses for some of our most popular intellectual property (IP) cores (shown in Table 1) in the Intel® FPGA IP Base Suite, which is free with the Intel® Quartus® Prime Software and Intel Quartus Prime Pro Edition Software.

If you have an active license for the Intel Quartus Prime Software or Intel Quartus Prime Pro Edition Software, simply log on to the Self-Service Licensing Center to generate your license file and take advantage of the IP Base Suite.

The IP Base Suite is also installed with the Intel Quartus Prime Lite Edition Software. Full licenses to this suite are purchased separately. Contact your local sales representative to learn more.

Table 1. Intel FPGA IP Functions in the IP Base Suite

Intel® FPGA IP Functions

Description
FIR II Over 12 major finite impulse response (FIR) filter architectures for use across multiple digital signal processing (DSP) applications. Generates bit-accurate and clock-cycle accurate models developed in Verilog, VHDL, and MATLAB, which provides coefficient generation. Provides significantly lower FPGA resource usage and increased performance than the original FIR Compiler for most FIR filter implementations. FIR filter performance, internal pipelining, and logic resource usage are optimized based on user inputs.

IP Base Suite Questions and Answers

The following are answers to frequently asked questions about the Intel FPGA IP Base Suite:

The Intel FPGA IP Base Suite is a set of perpetual licenses for full-featured versions of some of the most popular Intel FPGA intellectual property (IP) cores. See the above Table 1. Intel FPGA IP Functions in the IP Base Suite to find a full list of included cores.

Full licenses for the entire Intel FPGA IP Base Suite come free with purchase of a license for the Intel Quartus Prime Standard Edition or Intel Quartus Prime Pro Edition software. Users of other Intel Quartus Prime software products, such as Lite Edition or Development Kit Edition, can still access Intel FPGA IP cores in the IP Base Suite by using Intel's unique hardware evaluation feature called Intel FPGA IP Evaluation Mode or by buying a license to the IP Base Suite.

The cores in the IP Base Suite are part of the Intel FPGA IP Library. Intel FPGA IP download and installation have been integrated with the Intel Quartus Prime software. After you have downloaded any version of the Intel Quartus Prime software from the Download Center, you can immediately begin working with all Intel FPGA IP cores including the IP Base Suite.

If you are not a user of the Intel Quartus Prime Standard Edition or Intel Quartus Prime Pro Edition software, or if you are a user of other the Intel Quartus Prime software products, such as Lite Edition or Development Kit Edition, you can order a seat license to the Suite. After you have ordered the license, simply visit Intel’s licensing website to create or update your license file:

Visit our licensing center

Select Get My License File

Yes! You should have received licenses for each of these cores with your Intel Quartus Prime software license file. If you are unsure of your license, simply follow the instructions above, visit Intel’s licensing website, and update your license file.

The free Intel FPGA IP Evaluation Mode feature allows you to generate models, simulate them using industry-standard simulators, and generate time-limited FPGA program files. You must purchase a valid license when you decide to go to full production. The free perpetual IP Base Suite licenses allow you to use the IP cores for full production purposes.

Complete details are available on Intel's IP Base Suite licenses web page.

If you already downloaded a license for the IP Base Suite, you can continue to use the older core versions that were released at that time indefinitely. To use the latest IP Base Suite cores, and the new IP cores added to the IP Base Suite, you must either purchase a renewal license for the IP Base Suite (IPSR-BASE) or renew your Intel Quartus Prime software subscription.
