Overview

The Web Neural Network (WebNN) standard unifies AI PC application development, accelerating CPUs, GPUs, and NPUs efficiently.

AI PCs are increasingly enabling new capabilities for portability and local AI implementations. Learn how the WebNN API is standardizing interactions with CPUs, GPUs, and NPUs to gain optimal performance and predictable behavior.

With the release of Intel® Core™ Ultra processors, developers can innovate freely in a local environment without the need for cloud services and experience optimal flexibility for local AI projects.

Topics covered in this workshop include:

  • Learning the fundamentals of WebNN and the advantages that this standard offers
  • Understanding the benefits that come from WebNN to create applications
  • Previewing a WebNN application in action
  • Participating in a hands-on workshop to create a WebNN AI application
