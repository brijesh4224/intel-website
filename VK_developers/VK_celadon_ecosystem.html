Independent Software Vendors (ISV) and Operating System Vendors (OSV) for Celadon

Celadon provides an ecosystem for running Android* on top of powerful hardware from Intel. Like the Android Open Source Project (AOSP), bringing Celadon to a new device may require a certain level of expert knowledge to solve common challenges with setting up the initial virtual environment and tailoring the source code for a specific product. Choose from the following independent software vendors and operating system vendors who can enable and support your Celadon-based product.

Esper

Esper Foundation for Android is enterprise-grade AOSP for dedicated devices. Esper Foundation provides the reliability, stability, and security your devices require through long-term support and regular updates. Esper Foundation is always up-to-date with the latest patches and operating system versions delivered over-the-air. Built for x86 on Intel® architecture, it gives you the freedom to take advantage of the full range of Android’s massive hardware ecosystem.

Whether you're starting with bare metal, require virtualization, or already managing an Android device fleet, Esper Foundation is ready to meet your needs today and help you scale tomorrow. With Esper Foundation, you can experience truly touchless provisioning, test new environments with dual-boot, and gain the advantages of fully remote device access. Esper supports both bare metal and virtualized Celadon deployment.

Thundersoft*

ThunderSoft* provides operating system technologies, superior products, and solutions. As experts in mobile, IoT, automotive, and enterprise, ThunderSoft can fully support Celadon open source projects that run Android applications in containers or virtual machines. ThunderSoft also supports Celadon-based system development and validation, assists Intel to speed up the development of Android*-based innovation projects, improves the development efficiency of x86-based hardware platforms, and promotes different application scenarios and workloads.

Android* Smart Screen Solution
This solution is based on 11th generation Intel® Core™ processors (formerly code named Tiger Lake), which supports high-resolution display.

Android Retail Solution on Intel® Products Perform development and testing to customize and deliver Android operating systems for one retail point-of-sale products including integrated and external devices.

Emteria

Emteria helps enterprises to overcome challenges with AOSP and provides dedicated support for deploying Celadon in professional environments in a simplified way. Automated continuous integration and continuous deployment (CI/CD) pipelines are designed for building customized Android and rolling over-the-air operating system security updates for AOSP and Celadon-based products. This way, the latest version of Celadon is always available to download for free using the emteria.OS installer. The download package contains all components needed to start Celadon from a USB flash drive, allowing users to test the Celadon project on Intel® platforms quickly. In addition, the Emteria build infrastructure automatically enriches the Celadon project with additional tools and features for industrial use, like kiosk mode, integrated device management, automatic over-the-air security updates, remote screen mirroring, and monitoring device fleets in real time.
