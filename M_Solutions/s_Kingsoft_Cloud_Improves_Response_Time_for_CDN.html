Kingsoft Cloud Improves Response Time for CDN

3rd gen Intel® Xeon® Scalable processors with enhanced crypto acceleration help move content distribution to the edge.

At a Glance:

  • Kingsoft Cloud is a top cloud provider that has developed a comprehensive suite of cloud computing services, including CDN services that distribute user content to the edge to avoid network congestion and improve overall website response and availability.

  • New 3rd gen Intel® Xeon® Scalable processors with enhanced crypto acceleration helps alleviate the Internet Transport Layer Security handshake bottleneck and results in being able to serve 2.3 times more HTTPS requests, improving response time and overall service.1
