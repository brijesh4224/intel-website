About this offer

BittWare’s IA-780i is an Intel® Agilex™ 7 I-series FPGA card designed to address security challenges for data center, networking and edge compute workloads. The FPGA features optional hardened, high-speed cryptographic engines. The standard-height, 3/4-length card provides a balance of I/O and memory using the Intel Agilex® chip’s unique tiling architecture with two QSFP-DDs, DDR4 SDRAM SODIMMs, and PCIe Gen5 x16 with CXL support for a variety of applications. The IA-780i has support for Intel® oneAPI™, which enables an abstracted development flow for dramatically simplified code re-use across multiple architectures.
