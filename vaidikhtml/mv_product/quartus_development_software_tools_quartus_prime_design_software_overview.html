What's New in Version 24.3

Read the latest release notes

Agilex Device Support
  • Added support for Agilex 5 FPGA D-Series devices
  • Additional support for Agilex 5 FPGA E-Series devices
  • Additional device support for Agilex 7 FPGAs F, I, and M-Series
Enhancements
  • Enhanced the node-finder search filters to include various interface types for faster search queries
  • Enhanced the NoC view in Interface Planner, including:
    •  More accurate architecture display
    • Visualization of links, switches, and congestion
  • Enhancements to Nios V Processor:
    • Branch prediction support in Nios V/g core
    • Lockstep support in Nios V/g core
    • ECC Full support in Nios V/g core
    • Improved performance numbers of Dhrystone and Coremark benchmarks
    • Reduced resource utilization of Nios V/m & Nios V/c core
  • Enhancements to Advanced Link Analyzer:
    • Updated F-Tile FHT and FGT transceiver support.
    • Enhanced Platform/Simulation Engine

 

Other Features and Improvements
  • Machine-learning-based estimation of post-synthesis area
  • Extended no-cost licensing for various IP, including Nios V. Includes auto-fetch from within Quartus
  • Cycle-accurate models for simulation of NoC (HBM only)
  • Extension of pre-compiled components with pre-compiled simulation libraries, reducing compile time
  • Compile time, memory, and runtime improvements
  • Board IP Presets for DisplayPort, PCIe+PIO, and HBM+NoC
  • Improvements to Quartus Exploration Dashboard
  • Quartus GitHub integration with many new example designs      
  • Beta Docker Container Support

What's New for FPGA IPs in Version 24.3

Agilex™ 5 FPGA D-Series IP Features

Video and Vision Processing:

  •  VVP Textbox IP for real-time adjustments in visual data processing
  •  MIPI D-PHY and MIPI CSI-2 support up to 3.5G
  •  MIPI D-PHY with 4K video support for advanced imaging systems

 

Compute Interconnect:

  •  PCI Express Gen4 x8 protocol stack with VirtIO and SR-IOV for low-latency, high-performance computing

 

External Memory Interface:

  •  Support for DDR4/5 and LPDDR4/5 memory for fast data processing

 

Communications Interconnect:

  •  JESD204B/C for high-speed data conversion up to 17G, CPRI, and eCPRI with MAC capability enhancing communication efficiency
Agilex™ 7 FPGA IP Features

Compute Interconnect (PCI Express and CXL):

  •  In-system PCIe Eye Scan feature for real-time diagnostics without JTAG
  • MCDMA IP Improvements
  • CXL 2.0 with Global Persistent Flush and RAS support for robust data management and access control

 

Network Interconnect:

  •  Introduction of High-Speed FHT Equalization Tool.
  • Mixed FHT and FGT topology for efficient networking.
  • New example designs for PTP 1588 and Serial Lite IV with deterministic latency (up to 12 lanes).

 

Communications Interconnect:

  •  Apollo board support for JESD204 IP, enabling high-speed data conversions for military applications.

Frequently Asked Questions About Quartus® Prime Design Software

Quartus® Prime Software enables a fast path to turning Altera® FPGA, SoC, and CPLD designs into reality. It provides tools and features needed to help with every step from design entry and synthesis to optimization, verification, and simulation. Check out the Quartus Prime Software brochure for more details.

Quartus® Prime Pro Edition Software and Quartus® Prime Standard Edition Software require a paid license, but Quartus® Prime Lite Edition Software is license-free. A fixed-node subscription supports access to the Pro Edition and Standard Edition, was well as the Questa*-Altera® FPGA Edition with one year of maintenance. To obtain a license, go to the Intel® FPGA Licensing Support Center.

If you are ready to purchase a license or need advanced FPGA and SoC features—such as support for Agilex® 7, Stratix® 10, Arria® 10, and Cyclone® 10 LP device families—first purchase a paid license from the Intel® FPGA Licensing Support Center. To get started with the free version of Quartus® Prime Lite Edition Software or to download a licensed version, go to Quartus Prime Software downloads.
