Agilex™ 7 FPGA F-Series Development Kit (P-Tile and E-Tile)

The Agilex™ 7 FPGA F-Series development kit delivers a complete design environment that includes all the hardware and software within a PCI Express (PCIe) form factor you need to start taking advantage of the performance and capabilities available in Agilex™ 7 FPGA F-Series FPGAs with P-Tile and E-Tile for your design needs.

Learn more about  Agilex™ 7 FPGA F-Series devices ›

Use this Agilex™ 7 FPGA F-Series FPGA Development Kit to:

  • Develop and test PCIe 4.0 designs using the PCI-SIG-compliant development board.
  • Evaluate SoC features and performance using the Hard Processor System (HPS).
