Open Model Zoo

Expose your students to a wide range of pretrained deep learning models for common framework libraries, with links to their technical research documentation. Demonstration applications provide ideas for projects to build skills. The model downloader and automation tools make it easier and quicker to download any of these models and their datasets.

OpenVINO Toolkit & Jupyter* Notebook Tutorials

Access over 40 ready-to-run Jupyter Notebook tutorials in GitHub*. These are ideal for class experiments with deep learning inference using the OpenVINO™ toolkit. Teach your students how to use the OpenVINO toolkit API for high-performance AI inference. Modify and run code as part of learning exercises that include object detection, optical character recognition, text-to-speech, computer vision, NLP, and more.
