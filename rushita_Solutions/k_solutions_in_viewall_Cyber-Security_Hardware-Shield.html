Out-of-the Box PC Security

Intel® Hardware Shield, available on Intel® Core™ Ultra processors with Intel vPro® Enterprise Windows-based PCs provides comprehensive IT security for your business, including AI-powered threat detection. 1

As the frequency, complexity, and severity of cyberattacks and network security attacks continue to rise, trust Intel Hardware Shield to better protect your users and data. Included with Intel vPro Windows, this collection of security technologies helps defend against modern threats at each layer: hardware, BIOS/firmware, hypervisor, VMs, OS, and applications.

Below-the-OS Security at the Firmware and Hardware Levels

Improve network and cyber security with active measures and unique out-of-the-box features exclusive to Intel Hardware Shield, which includes advanced threat protection, application and data protection, and below-the-OS security.

  • Protect your users from threats like ransomware, cryptojacking, and software supply chain attacks with Intel® Threat Detection Technology (Intel® TDT), a trusted AI-enhanced security technology available with Intel Hardware Shield.
  • Help ensure your OS boots securely and launch your PCs in a trusted state with Intel Hardware Shield, which prevents malware from compromising the OS and offers hardware-based security capabilities that support a secure startup.

Learn more: Below-the-OS Security white paper

Extend Protections with Hardware-Accelerated Virtualization and Encryption

Boost IT security for virtualized environments with Intel vPro Enterprise for Windows, which extends hardware-rooted features to help protect user access credentials, workspaces, applications, and data in hardened enclaves.

  • Run virtual machines for security-based isolation while Intel Hardware Shield’s capabilities support application compatibility across different operating systems running on the same PC.
  • Accelerate virtualized security software like Windows Defender Credential Guard and Application Guard with Intel Hardware Shield’s virtualization capabilities to help protect against OS kernel-level malware and browser-based attacks.
  • Complement virtualization with hardware-based encryption to help protect data at every layer.

Learn more: App and Data Protections white paper

Help Detect and Respond to Advanced Threats

Mitigate extreme network security attacks like control-flow hijacking, ransomware, and crypto mining.

  • Protection shouldn’t come at the cost of productivity, so Intel® Threat Detection Technology (Intel® TDT) utilizes all three compute engines in Intel® Core™ Ultra processors, making independent security software faster and more effective while providing extra performance headroom for demanding business applications.
  • Help defend against attacks on system memory with Intel® Control-Flow Enforcement Technology (Intel® CET).

Enhance System Security at Every Start

Users are better protected whenever they start their PCs with Intel® Silicon Security Engine.

  • Improve system security with hardware-based authentication that helps ensure system-critical software has not been tampered with.
  • Intel® Silicon Security Engine protects the firmware that operates new power-management circuitry, plans to extend these protections to other parts of firmware are in development.

Add Trust to Your Client Systems

Ensure you can rely on your supply chain with the Intel® Trusted Device Setup (Intel® TDS) stamp of OEM attestation.

  • Help verify that new PCs have not been tampered with post-manufacturing before corporate security software is installed—even when shipped directly to end users working from home.
  • The Intel® TDS OEM remote health attestation service delivers a signed health report and a digital seal that captures events that could pose a security risk, if the device has been tampered with.
  • Determine if the PC has been tampered with by detecting chassis intrusion, firmware or BIOS changes, storage drive replacement, boot while in transit or other changes that break the digital seal.
