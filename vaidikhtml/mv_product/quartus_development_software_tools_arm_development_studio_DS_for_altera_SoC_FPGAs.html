What's New

Intel has migrated to the new Arm* Development Studio for Intel® SoC FPGA (Arm* DS for Intel® SoC FPGA) and support for Arm* DS-5 has been dropped starting with Intel® Quartus® Prime Pro and Standard® software version 20.1. Developed in partnership with Arm, the Arm* Development Studio for Intel® SoC FPGA (Arm* DS for Intel® SoC FPGA) is an end-to-end suite of tools for embedded C/C++ software development on any Intel® SoC FPGAs. Arm DS for Intel® SoC FPGA works in conjunction with the Intel® SoC FPGA Embedded Development Suite (SoC EDS).

Learn more about Intel® SoC EDS

The Arm DS for Intel® SoC FPGA is no longer installed as a part of SoC EDS and is a separate download.

Why use Arm DS for Intel SoC FPGA?

Code

Eclipse*-based IDE provides a great workbench with Arm's assembly editor and project management tools.

Build

Compile and build code for high performance or small footprint with included Arm Compiler 5 or 6.

Debug

Debug Bare-metal, Linux, and Android application code with low-cost Intel® FPGA Download Cable II.

Optimize

Optimize and make your Linux* or Bare-metal application more efficient with Streamline* and Trace.

Features

FPGA-Adaptive Debug

The FPGA-Adaptive Debug is a powerful feature that gives a system-wide view of the SoC, allowing you to cross-trigger between the hard processor system (HPS) and the FPGA with ease.

Program Trace

Unleash the power of the CoreSight* System Trace Macrocell (STM) Debug architecture to view timeline carts and heat maps for software that executes on target.

Multi-Core Debug

Debug multiple processor cores simultaneously and control them individually.

Application Performance Analysis

Visualize the performance bottlenecks in your embedded software using Streamline*. Support for Bare-metal, Linux* and Android* applications.

Getting Started

Arm* DS for Intel® SoC FPGA is license managed and available with a paid license. The license shipped with select Intel® SoC FPGA development kits includes 3 years of Intel support and maintenance from the date of purchase. It supports all Arm processor subsystems on Intel® SoC FPGAs and provides exceptional bare-metal debug and trace capabilities. Download and set up product license for Arm DS for Intel® SoC FPGA.
