Error Message: Computer Manufacturer Graphics Driver Detected

Are you seeing either of these error messages?

While using Intel® Driver & Support Assistant

ERROR: Installing this Graphics driver from Intel may remove customizations from your computer manufacturer (OEM).

During manual driver installation

ERROR: The driver being installed is not validated for this computer. Please obtain the appropriate driver from the computer manufacturer. Setup will exit.


Before attempting to resolve this problem

Before continuing, identify your Intel Core® Processor and its Generation. Based on which generation processor you have, you can find the resolution below:

Processors Why Solution
6th, 7th, 8th, 9th, 10th, 11th, & 12th Generation Intel® Core™ Processor Graphics, & related Intel Atom®, Pentium®, and Celeron® Processors This error is occurring on Intel® 6th Generation Core Processors or Newer Processors because the driver installed on the system is non-DCH.

Follow the steps in OEM Error Message Appears When Installing Intel Graphics Drivers on 6th Generation Intel Processors or Newer to resolve this issue.

Note: You will not receive this error if you have the latest DCH driver installed. Check if you have a DCH or non-DCH driver.
5th, 4th, 3rd, 2nd, and previous Generations Intel® Core™ Processor Graphics, & related Intel Atom®, Pentium®, and Celeron® Processors This error occurs on Intel® 5th Generation Core Processors or Older Processors when a driver installed from your computer manufacturer (OEM) prevents the system from updating from an OEM driver to a generic Intel driver.

We advise you to always get the latest driver from your computer manufacturer.

However, if your computer manufacturer no longer provides new drivers or you prefer to install a newer Intel generic driver, you can perform a clean installation of a generic Intel® Graphics Driver.

Caution: Be aware that you risk system instability, losing functionality, or causing other technical issues that OEM drivers seek to resolve or enable.
