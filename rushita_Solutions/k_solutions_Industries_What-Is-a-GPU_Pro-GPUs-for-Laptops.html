Ray Tracing Built-In

Graphics cards need to do more than push pixels to displays, they also need to accelerate the creation of beautiful, compute intensive imagery. Ray-tracing hardware technology in the built-in Intel Arc Pro GPU allows your compatible software to explore these new pro workflows. Ray tracing is typically thought of as only available on expensive add-in GPUs, but our latest built-in GPU reduces this barrier.

Multiple Large Displays with a Built-In GPU

The built-in Intel Arc Pro GPUs for laptop on select Intel Core Ultra Processors can support up to four ultra-large displays6 to enhance your workflow efficiencies.

16GB Minimum Memory Capacity

The demands professional software places on integrated and built-in graphics continue to increase with each new software release. To meet these demands, we specify Intel Arc Pro built-in GPUs must have a 16GB minimum memory capacity, offering lots of memory space for the built-in GPU to utilize within your favorite software. More memory allows you to take on bigger video projects, tackle larger assemblies, develop complex models, and work on detailed 4K imagery.

Full AV1 Support

Full AV1 hardware acceleration. Get 4.4x faster hardware encoding with Intel Arc GPU compared to software encoding.3
