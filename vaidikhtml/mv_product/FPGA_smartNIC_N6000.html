Intel® FPGA SmartNIC N6000-PL Platform

High Performance Agilex™ FPGA-based SmartNIC Platform providing 2x100GE connectivity designed to support communications and networking workloads, such as 4G/5G Virtualized Radio Access Network (vRAN), Accelerated Virtual Cell Site Router (vCSR), Cloud-Native Contrail Networking (CN2), 5G User Plane Function (UPF) offload, Professional Media over Managed IP Networks, and more.

Read the product brief ›

Overview

  • Intel’s 3rd generation SmartNIC card provides 2 x 100Gbps Ethernet connectivity in a FHHL PCIe form factor card.
  • Integrated support for IEEE 1588v2 PTP and SyncE for O-RAN, Broadcast and other markets that need timing synchronization support. The on-board OCXO provides timing accuracy and extended holdover.
  • On-board Intel E-810 Ethernet Controller provides easy system integration with no changes to application software
  • N6000-based SmartNICs come in two configurations:
    1. Intel® FPGA SmartNIC N6000-PL Platform (with an on-board Ethernet controller): PCIe 4.0 x8 lanes go to the FPGA and x8 lanes go to the Ethernet controller.
    2. Intel® FPGA SmartNIC N6001-PL Platform (without an Ethernet controller): PCIe 4.0 x16 lanes go to the FPGA.
