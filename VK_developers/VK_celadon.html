Supported Platforms

Celadon is designed for all Intel® platforms that are enabled for Linux*. Its solutions are enabled on:

  • Android 14 with a 6.1 LTS kernel on 13th generation Intel® Core™ processors and higher

For previous Android releases, see the release notes

Supported Components

Celadon supports the following hardware components. For details on the latest additions, see the release notes

AOSP
  • Based on Google* upstream AOSP
  • Android 14 with API Level 34 and Target Level 8
  • 90+% HALs support AIDL
Graphics
  • MESA 3D Graphics Library with OpenGL* ES v3.2 and Vulkan* v1.3 support
  • Gralloc graphics memory allocator v4.0
  • HDMI display support
  • Hardware composer (HWC) v2.4 display support
Audio
  • USB digital audio playback
  • Analog audio (MP3 and Advanced Audio Coding [AAC]) playback on a stereo headset
  • Audio mediation through a host
Connectivity
  • Wi-Fi* 802.11a/b/g/n
  • Wired Ethernet
  • Dual-mode Bluetooth® technology v4.2
Security
  • Trusty TEE
  • Hardware-backed Keymint and Gatekeeper
