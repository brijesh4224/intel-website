NoLoad Transparent Compression

Category:

Software and IP Cores: FPGA Accelerator Functions: Storage and Compression

About this offer

Eideticom’s NoLoad® computational storage solutions are accelerating data center infrastructure, enabling greater scalability, increased performance and dramatically lowering cost. NoLoad provides hardware-based transparent compression that enables increased storage capacity (lower $/TB) without sacrificing performance. Eideticom's NoLoad offloads storage functions onto accelerators enabling line-rate compression, improving CPU utilization, and reducing memory bandwidth pressure. This results in 50x increased throughput performance at 20% CPU utilization while maintaining compression ratio. Other benchmarks have resulted in 5x increased storage capacity while maintaining all other performance metrics using postage SQL.
