Explore AI

Intel® Distribution of OpenVINO™ Toolkit

Use this open source toolkit for high-performance deep learning inference. The toolkit complements your STEM courses that teach computer vision, natural language processing (NLP), and recommender system design using many popular AI framework libraries like TensorFlow* and PyTorch*.

Learn More

Intel® Developer Cloud

Learn through curated experiences, including Jupyter* Notebook tutorials and real-world use cases. With free access, get hands-on experience with Intel hardware and software in a cloud environment that comes preinstalled with Intel-optimized frameworks, tools, and libraries.

Get Started

Jupyter* Notebook

Access over 40 ready-to-run Jupyter* Notebook tutorials in GitHub* to experiment with deep learning inference using the OpenVINO toolkit. Learn how to take advantage of the OpenVINO toolkit API for high-performance AI inference. Practice with object detection, optical character recognition, text-to-speech, computer vision, and NLP.

Intel® AI Developer Program on YouTube*

Watch videos that help you create projects on different hardware options using a wide range of IDEs and programming languages, including Arduino*, C/C++, JavaScript*, Node.js*, Java*, Python*, and more.
