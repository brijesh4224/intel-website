Strengthen Your Endpoint Security Solutions

Cyberattacks continue to evolve novel techniques to hide from security software deployed on endpoint PC fleets. Traditional software-only approaches are failing to stop attacks. Ransomware attacks are up to a record $ 1 billion in payouts in 20231, 75% of attacks use stealthy file-less attack methods, and the breakout time to move across a PC fleet is down to only 62 minutes.2

Intel Threat Detection Technology has delivered an innovative approach that is able to fingerprint malware as it attempts to execute on the CPU microarchitecture. This method is immune to traditional malware cloaking or obfuscation techniques and adds a new lens that helps increase the efficacy and performance of security software.

By pairing the right Intel PC hardware with Intel® TDT-optimized security software IT and Sec Ops can better protect PC fleets from ransomware, cryptojacking, file-less attacks to memory, and software supply chain attacks that attempt to infect valid software applications.

Help Detect Advanced Threats with Powerful AI Tuned for Your PC

Intel® TDT uses CPU telemetry and Machine Learning (ML) algorithms to profile and detect malware, such as ransomware and cryptojacking, improving monitoring and security performance at the hardware level. In a recent study, Intel® TDT detected 93 percent of top ransomware variants from the silicon sensor, bolstering the EDR’s overall detection efficacy by 24% over software alone.3

Accelerate Security Software Out of the Box

Intel® TDT has been pre-integrated by leading security ISVs to accelerate security workloads on Intel® Core™-based PCs. Intel's CPU, GPU, and NPU deliver AI accelerators to ensure a performant user experience while increasing endpoint threat protection capabilities.4

Enhance Detection of Fileless Malware

By offloading advanced memory scanning (AMS) workloads from the CPU to the Intel® integrated GPU, Intel® TDT enables software-based security solutions to scan, deeper, and more frequently. This helps software solutions uncover file-less attacks sooner before they can execute attack payloads while ensuring minimal impact on the user experience.

Help Discover Zero-Day Attacks

CPU behavior monitoring supports real-time discovery of zero-day attacks and new variants without stopping to develop signature-based detection methods.

Intel vPro®5 is the only business platform with built-in hardware security capable of detecting ransomware and software supply chain attacks.6
