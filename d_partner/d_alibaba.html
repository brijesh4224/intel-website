Elastic and Secure Compute

Scale up to hundreds of thousands of vCPUs in minutes for a single customer in a single region. With numerous general and advanced compute instances based on Intel® Xeon® Scalable processors, Alibaba Elastic Compute offerings deliver high availability and reliability built in with security.

Key features:

  • Scenario-based instance types: Select your service based on individual workload requirements for high efficiency and utilization.
  • Meet SLAs: Alibaba Cloud offers a 99.975% availability commitment for individual instances and 99.995% for multiple zones in the same region.
  • Flexible billing: Choose between subscription, pay-as-you-go, or reserved instances to help optimize your resources and budget.

Alibaba PolarDB

Support business-critical database applications with this cloud-native relational database service, compatible with MySQL, PostgreSQL, and Oracle.

Key features:

  • Decoupled compute and storage: Optimize costs with dedicated compute nodes for executing SQL read/write and dedicated storage nodes for data storage.
  • Stability and security: Active/standby switchovers help prevent data loss and inconsistency, combined with read-only replicas that can be created in minutes.
  • Developer support: Always-free PolarDB offers up to 50 GB of storage for developers to experiment and optimize their applications.

Learn more about Alibaba PolarDB

Alibaba HPC Cloud Instances

Built with a foundation of Intel® Xeon® Scalable processors, Alibaba Super Computing Cluster (SCC) instances deliver high-quality network performance, high bandwidth, and low latency for the most demanding workloads.

Key features:

  • Maximum performance with up to 96 vCPU cores, up to 512 GB memory, and up to 8 GPU cards within a single instance.
  • High-speed interconnect with Remote Direct Memory Access (RDMA) technology.
  • Secure and elastic, capable of deploying new servers in minutes and resizing as needed.
