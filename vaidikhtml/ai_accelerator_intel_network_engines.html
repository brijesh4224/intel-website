Accelerate Data Transmission and Reduce Latency

The growing demands on network bandwidth and performance continue to challenge developers whose solutions must support emerging usage models within today’s budget constraints.

Intel® Network Engines include a set of workload-specific accelerators built into Intel® Xeon® Scalable processors to deliver significant network performance improvements. These integrated capabilities remove the need for external accelerators to elevate performance while helping to reduce costs and simplify solution designs.

Data Compression

Intel® QuickAssist Technology (Intel® QAT)

Boost performance of demanding network workloads so systems can handle more data and serve more clients. Use Intel® QAT to offload critical compression, decompression, encryption, and decryption processes from the CPU core and accelerate those operations to save valuable compute resources.

vRAN Efficiency

Intel® vRAN Boost

Simplify the development and operation of flexible, scalable, energy-efficient virtualized radio access network (vRAN) and Open vRAN solutions. With excellent performance per watt, Intel® vRAN Boost can help you conserve energy and improve total cost of ownership.

Learn how Intel® vRAN Boost works

Queue Optimization

Intel® Dynamic Load Balancer (Intel® DLB)

Balance and distribute network traffic dynamically and manage queues intelligently across CPU cores. Intel® DLB helps improve performance and throughput for data streaming, packet processing, gateway handling, and other compute-intensive networking tasks.

Learn how Intel® DLB works

Streaming Content

Intel® Data Streaming Accelerator (Intel® DSA)

Optimize transmission and speed up the movement and transformation of streaming data, which are common operations in high-performance networking, storage, persistent memory, and data processing applications. Intel® DSA offloads these and other compute-intensive operations from the CPU to minimize network latency while reducing the cost and complexity associated with discrete accelerator hardware.

Learn how Intel® DSA works
