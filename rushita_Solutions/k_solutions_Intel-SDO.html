How Zero-Touch Provisioning Works

View how Intel SDO software tools implement the onboarding protocols to securely provision a device to a device management system.

An Open & Interoperable Industry Standard

Intel SDO provides a fast and more secure way to onboard any device to any device management system. This proven method is in broad adoption by device management system vendors. To help set the unified standard for device provisioning, these use cases and protocols have been submitted to the FIDO* Alliance working group for IoT.
