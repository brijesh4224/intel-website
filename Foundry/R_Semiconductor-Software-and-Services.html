Move Swiftly by Integrating Cores and Accelerators Using oneAPI

Get faster time to market and enable more robust solutions by moving portably and efficiently using Intellectual Property (IP) and software reuse to minimize new investments. oneAPI is an open, cross-industry, standards-based, unified, multiarchitecture, multi-vendor programming model that delivers a common developer experience across accelerator architectures – for faster application performance, more productivity, and greater innovation.

Accelerate Time to Market

The oneAPI initiative is based on open standards for cross-vendor compatibility and enables a ‘write once, run anywhere’ approach by delivering a unified compute software development environment across CPU and accelerator architectures. Learn more about oneAPI.

The oneAPI Specification is now part of the Unified Acceleration (UXL) Foundation.

Intel provides the Intel® oneAPI Base Toolkit as a core set of tools and libraries for developing high-performance, data-centric applications across diverse architectures.
