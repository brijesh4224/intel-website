Can Intel® Stratix® 10 GX FPGA Development Kit support PCIe gen3x16 for OpenCL™?

  • Description
  • Resolution
Environment

Quartus Edition

Intel® Quartus® Prime Pro Edition

Version Found: 19.1

Quartus Add-ons

Intel® FPGA SDK for OpenCL™ Pro Edition

Description

Due to limitations in support in OpenCL™ Board Support Package (BSP), Intel® Stratix® 10 GX FPGA Development Kit can only support up to PCIe gen3x8 for OpenCL™, although its PCIe hard IP can support PCIe gen3x16.

 

 

Resolution

To work around this problem, you can create a custom BSP for the Intel® Stratix® 10 GX FPGA Development Kit.

There is no current plan to add PCIe Gen3x16 to the OpenCL™ Board Support Package (BSP) for the Intel® Stratix® 10 GX FPGA Development Kit.
