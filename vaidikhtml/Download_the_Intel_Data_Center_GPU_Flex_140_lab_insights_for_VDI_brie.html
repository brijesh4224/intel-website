Description

Intel contracted the Futurum Group Labs to evaluate the performance, and price-performance of multiple HCI configurations supporting VMware Horizon VDI deployments running clusters of servers using Intel CPUs and GPUs, utilizing VMware vSphere together with vSAN storage. Adding GPUs to configurations is an effective way to enable higher graphics performance for VDI users that need it, while offloading CPU resources for other VDI users. Get the facts.
