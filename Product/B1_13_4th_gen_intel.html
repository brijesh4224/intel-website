4th Gen Intel® Xeon® Scalable Processors for IoT Edge

Designed to deliver the most powerful compute for demanding edge workloads, 4th Gen Intel® Xeon® Scalable provides improvements in performance, faster memory speeds with DDR5, greater I/O support with 80 lanes of PCIe Gen 5.0 and offers the most built-in accelerators in the market.

Build efficient IoT solutions for video analytics, industrial machine vision, edge control systems, and other use cases with Intel® Xeon® Scalable processors. The 4th generation of Intel® Xeon® Scalable processors introduces new Intel® Advanced Matrix Extensions (Intel® AMX), a built-in AI acceleration engine. Intel® AMX delivers 3.01x increased AI inference performance for image classification1 and 4.25x higher AI inference performance for object detection2 over the previous generation, along with new capabilities to accelerate AI modeling and inference for edge servers, industrial machine vision, and video analytics applications.

Edge applications benefit from the long-life availability3 and industrial-grade features4 in a wide range of power and performance envelopes, with hundreds of repeatable market-ready solutions that enterprises can deploy quickly and at scale.

Key Features on 4th Gen Intel® Xeon® Scalable Processors

The new microarchitecture of the 4th Gen Intel® Xeon® Scalable processors is designed to meet the needs of dynamic and demanding workloads at the edge. These processors offer performance, bandwidth, and built-in acceleration for edge applications such as secure video analytics, factory automation, patient care and research for healthcare, and analytics for retail, banking, and hospitality.

More Cores, More Memory, More I/O

Compared to the prior generation, 4th Gen Intel® Xeon® Scalable processors increase performance by 1.33x5 with up to 52 cores or sockets6 on SKUs built for IoT edge use cases. They also improve socket scalability with support for single-socket or two-socket platforms.7

This latest generation includes up to eight channels of DDR5 memory per CPU with support for up to 4800 MT/s and a maximum of 16 DIMMs per socket.

Built-In AI Acceleration

Accelerate AI inferencing workloads and video analytics for object detection and image classification with Intel® AMX and support for the Intel® Distribution of OpenVINO™ toolkit. AI acceleration on the 4th Gen Intel® Xeon® Scalable processors is optimized for industry-relevant frameworks and libraries while Intel® AMX accelerates a variety of deep learning inference and training workloads with BF16 and INT8 datatype support.

More Control and Virtualization

The 4th Gen Intel® Xeon® Scalable processor offers better control over CPU performance with Intel® Speed Select Technology (Intel® SST) which is ideal for consolidating workloads. Memory bandwidths are used by applications such as virtual machines (VMs), and containers with Intel® Resource Director Technology (Intel® RDT). Intel® RDT enables monitoring and control over shared resources, while Intel® Scalable I/O Virtualization (Intel® Scalable IOV) provides efficient resource allocation across VMs, containers, and functions.

Resilience and Security

Protect your data and your platforms with Intel’s reliable security8 and resilience features available on 4th Gen Intel® Xeon® Scalable processors. Create trusted enclaves within applications with Intel® Software Guard Extensions (Intel® SGX) and encrypt data in memory with Intel® Total Memory Encryption (Intel® TME). Meanwhile, Intel® Platform Firmware Resilience (Intel® PFR) monitors system bus traffic, verifies platform integrity, and restores corrupted firmware.

Topline Specifications

  • Intel® Advanced Matrix Extensions (Intel® AMX)
  • Intel® Speed Select Technology (Intel® SST) for granular control over CPU performance
  • Up to 8 channels DDR5 with up to 4,800 MT/s and up to 16 DIMMS per socket
  • New support for Compute Express Link 1.1 (CXL 1.1) on up to 4 devices per CPU
  • Up to 80 lanes PCIe 5.0 and 4.0
  • Hardware-enhanced security with Intel® Software Guard Extensions (Intel® SGX) and Intel® Total Memory Encryption (Intel® TME)
  • Single- or dual-socket support for select IoT SKUs7
