Graphics Trace Analyzer

Evaluate workload performance across the CPU and GPU, ensuring your game uses the full processing power of Intel® architecture.

Features

  • Explore queued GPU tasks.
  • Examine CPU thread utilization.
  • Pinpoint CPU and GPU activity based on captured platform and hardware metrics.
  • Use the timeline to review tasks, threads, Microsoft DirectX*, Vulkan*, and GPU-accelerated media applications.
  • Correlate events on the same timeline and get further insight into performance details using the Events Tracing for Windows (ETW) Providers.
  • Understand full system latency with virtual reality (VR) compositor events.
  • Toggle event domains to collect only the data you need, reducing the application workload.
