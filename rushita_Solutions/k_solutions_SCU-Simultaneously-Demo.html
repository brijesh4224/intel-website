Configuring Intel® Embedded Server RAID Technology 2 Using AHCI and SCU Simultaneously Demo

Content Type: Install & Setup | Article ID: 000006197 | Last Reviewed: 02/27/2025

Complete the steps to configure Intel® Embedded Server RAID Technology 2 for AHCI and SCU simultaneously.

Configure Intel® Embedded Server RAID Technology II Using BIOS Console Demo for Server Products Using Intel® Server Board S2600CO2 as an example, the board has the following:

  • Two white ports of 6 GBPS (SATA 0 and SATA 1) from the AHCI controller. The white ports are SATA by default, and cannot be changed to SAS.

  • Eight blue ports (SATA/SAS_0 to SATA/SAS_7) from SCU controller. But, you can change the blue SATA ports (default), to SAS by connecting a RAID upgrade Key. See the How to Configure Intel® Embedded Server RAID Technology 2 Using AHCI and SCU Simultaneously section in the Technical Product Specification.

  • Four hard drives. Two hard drives each of 6 Gb/s and 3 Gb/s, and No Upgrade Key. There are 4 blue ports that are active: SATA/SAS_0, SATA/SAS_1, SATA/SAS_2, and SATA/SAS_3.

Steps:

  1. Connect two hard drives of 6 Gb/s in the White ports, SATA 0 and SATA 1.
  2. Connect two hard drives of 3 Gb/s in the blue ports, SATA/SAS_0 and SATA/SAS_1.
  3. Power on the Server. When prompted, press F2 to enter BIOS.
  4. Disable Quiet boot from the Main Option.
  5. Navigate to Advanced > Mass Storage Controller Configuration.
  6. Change AHCI controller configuration to RAID.
  7. Change AHCI capable SATA controller to LSI ESRT.
  8. Change SAS/SATA controller Configuration to LSI ESRT.
  9. Press F10 to Save the settings and restart the server.
  10. When the system restarts, you are prompted to press CTRL+E option to Enter Embedded Server RAID Technology. This option is for the hard drives that are connected to the AHCI controller.
  11. When you see the Management Options, click Configure.
  12. Go to New Configuration. When asked if you want to Proceed, select Yes and select the hard drives which have Status Ready with your mouse. Press the Space bar on the selected hard drives. This action changes the status to Online.
  13. Press F10, press space bar to get SPAN-01. Press F10. Select the RAID level. Click Accept. Press space bar.
  14. Press ESC once, to get Save Configuration, click Yes.
  15. Press ESC until you get Management Options again.
  16. Click Initialize. Press space bar to select the Virtual Drive.
  17. Press F10 to start the initialization process.
  18. When completed, press ESC to Exit and click Yes. Press CTRL+ALT+DEL to start.
  19. After restarting, you see CTRL+E prompt. Skip this, and you are prompted to press CTRL+E again for the AHCI Controller.
  20. Follow the same steps used for the AHCI controller (steps 10-18), to configure RAID in ESRT-2 for ACHI Controller.

This completes the guide on how to configure RAID on both the onboard SATA controller.

Note These instructions might also be applicable to Intel® Server Boards that utilizes the following Intel® Chipsets: C600, C222, C224, and C226.
