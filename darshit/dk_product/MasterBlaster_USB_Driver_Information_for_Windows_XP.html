MasterBlaster™ USB Driver Information for Windows XP

  • Product Discontinuance Notifications
  • Driver Installation - Quartus® II 2.1 and Later
  • Driver Installation - MAX+PLUS® 10.2 and Later
  • Related Information

Product Discontinuance Notifications

Intel has discontinued the Quartus® II and MAX+PLUS® II Software. More details are available at Quartus® II and MAX+PLUS® II Software product discontinuance notifications and list of discontinued additional software.

Windows will prompt you to install the MasterBlaster universal serial bus (USB) cable driver the first time the MasterBlaster programmer is plugged in.

Quartus® II 2.1 and Later - Driver Installation

When you plug in the MasterBlaster USB cable, Windows XP will automatically recognize it and prompt you for the location of the driver.

  1. Browse to the <Path to Quartus II software>\drivers directory.
  2. Select the mblaster.inf file and click OK.
  3. Set up programming hardware in the Quartus II software to complete your installation.

MAX+PLUS® 10.2 and Later - Driver Installation

When you plug in the MasterBlaster USB cable, Windows XP will automatically recognize it and prompt you for the location of the driver.

  1. Browse to the <Path to MAX+PLUS II software>\Drivers\i386 directory.
  2. Select the mblaster.inf file and click OK.
  3. Set up programming hardware in the MAX+PLUS II software to complete your installation.
