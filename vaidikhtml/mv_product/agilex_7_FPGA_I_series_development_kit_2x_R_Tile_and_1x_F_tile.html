Agilex™ 7 FPGA I-Series Development Kit (2x R-Tile and 1x F-Tile)

The Agilex™ 7 FPGA I-Series development kit delivers a complete prototyping and reference platform for designs needing PCI Express (PCIe) 5.0, Compute Express Link (CXL), or 400G Ethernet connectivity.​

Learn more about Agilex™ 7 FPGA I-Series devices ›​

Use this Agilex™ 7 FPGA I-Series FPGA Development Kit to:​

  • Develop and test PCIe 5.0 x16 (up to 32 GT/s per lane) designs using the PCI-Express form-factor compliant development board using applicable IP from Intel or third-parties*.
  • Develop and test CXL v1.1/2.0 x16 designs using the applicable IP from Intel or third-parties.​
  • Develop and test high-speed Ethernet (up to 400G total bandwidth) using the applicable IP from Intel or third-parties.
  • Connect to daughter-cards via MCIO connector/cable for additional connectivity or memory options (not included in purchase. Contact Intel for more details.).

* The Intel® FPGA CXL IP is licensed separately. Contact your local Intel salesperson for IP variant options and pricing details.
