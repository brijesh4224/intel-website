Intel® Xeon® D Processor

With better performance, a higher core count, and more flexible connectivity, Intel® Xeon® D processors offer speed and efficiency for edge and network solutions where density and low power are essential.

Key Features

• Improved performance at a similar TDP.

• Workload-optimized performance in space and power-constrained environments, from the cloud to the edge.

• Innovative, system-on-chip processors support high-density, single-socket network, storage, and cloud-edge computing solutions with a range of integrated security, network, and acceleration capabilities.

• A higher core count and more flexible ethernet connectivity, accomplished on the same platform as Intel® Xeon® D Processors (codename Ice Lake-D), help secure customer investments and minimize validation effort.

• Now equipped with Intel® Speed Select Technology (Intel® SST), featuring support for various Turbo frequency profiles and Base Frequency splits, enabling optimized power budgeting for certain cloud and network workloads.

Product Highlights of D-1800/2800

• Introducing a new 22 core "top of the line" SKU, with 2 more cores than the D-2700

• Platform and Drop-in compatible with D-1700/2700

• New 2x100G ethernet connectivity on D-1800

• Select SKUs available with Intel® SST-TF

Product Use Cases

The D-1800/2800 is a highly integrated SoC designed on a BGA substrate, enabling a lower TDP delivering the best-in-class feature set per watt for Xeon D SoCs. Based on the highly integrated functionality that D-1700/2700 delivered for the development of compact solutions for use in indoor, outdoor, and ruggedized environments.

Server on A Card

A compact computing solution combines a server's essential components, including processor, memory, storage, and networking, onto a single card or board.

Edge Servers

Specialized compute resources running the processing close to the network edge, reducing bandwidth and enabling real-time decision-making, are typically found in a local data center or IoT devices.

Network Appliances

The design methodology and integrated functionality comprising of high-end compute, Ethernet and Security enables various solutions to be designed including Network Appliances like Enterprise routers, switches for the cloud and network edge, uCPEs as well as storage appliances.
