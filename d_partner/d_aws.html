Instance Types

Intel technologies are the hardware infrastructure powering the majority of AWS instances. These instances span a massive array of demands. Instances can be scaled and optimized with the right mix of resource capacity—CPU, memory, storage, and networking—to meet each customer’s specific workload needs.

General Purpose

Intel-based general purpose AWS instances offer balanced cloud infrastructure to suit a wide variety of workloads. These instances are ideal for applications that use compute, memory, and networking in equal proportions, such as web servers and code repositories.
