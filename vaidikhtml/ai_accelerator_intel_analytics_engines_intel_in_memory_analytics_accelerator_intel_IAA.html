Accelerate Development with Faster Data Analytics

Databases and data lakes continue to grow in size and complexity, multiplying the challenges for developers and analysts. Data processing—especially compression and decompression—of large datasets can strain computing resources, hamper development efforts, and slow down business decision-making.

Data-dependent organizations need a high-performance computing platform to speed development and generate business insights at a competitive pace.

Intel® Xeon® Scalable processors with built-in Intel® In-Memory Analytics Accelerator (Intel® IAA) address this need. Intel® IAA accelerates database queries and optimizes performance by offloading compression, decompression, and other compute-intensive tasks from the CPU. This purpose-built accelerator can reduce or eliminate the need for costly, add-in PCI cards or other supplemental hardware that typically require integration, creating data bottlenecks and consuming additional power.

Improve Data Analytics Performance

Intel® IAA can perform computationally demanding scanning and filtering of large datasets, freeing up the embedded CPU cores. With its hardware-based compression, Intel® IAA can accelerate data analytics performance and throughput while reducing memory bandwidth requirements.

Reduce TCO and Power Consumption

Intel® IAA boosts data processing efficiency so that the same server can handle more data. With fewer servers and cores needed, Intel® IAA can reduce power consumption and total cost of ownership while helping to improve sustainability.

Simplify Integration and Development

Intel® IAA is built into Intel® Xeon® Scalable processors helping to reduce or eliminate the need for discrete hardware, including expensive PCI cards. This helps to simplify integration and configuration as there are fewer components to worry about.

Harness Open Source Libraries and Tools

Intel provides development tools, open source libraries, plugins, drivers, and documentation to simplify activation of Intel® IAA in your applications. Our open source Intel® Query Processing Library (Intel® QPL) makes it easy to activate and enable Intel® IAA in new and current solutions, including those using popular open source databases and database management systems.
