ONNX* (Open Neural Network Exchange) Model and ONNX Runtime

ONNX* model is a machine learning model format, and ONNX Runtime is a cross-platform inference and training machine learning accelerator. It works with IntelÂ® platforms and allows developers to improve model performance while easily targeting multiple platforms. ONNX Execution Providers (EP) enable certain hardware acceleration technologies to run AI models. Intel platforms have two optimized EPs: the OpenVINO Execution Provider and the DirectML EP.
