Detailed Description

Purpose

The purpose of the Intel® Processor Diagnostic Tool is to verify the functionality of an Intel® microprocessor. The diagnostic tool checks for brand identification, verifies the processor operating frequency, tests specific processor features, and performs a stress test on the processor.

Note: The file name for both 32-bit and 64-bit executable is:

  • IPDT_Installer_4.1.9.41_32bit.exe
  • IPDT_Installer_4.1.8.40_64bit.exe

IPDT_4.1.8.40_src.zip is source code for the Intel® Processor Diagnostic Tool test modules.

See the Release Notes for installation instructions, supported hardware, what is new, bug fixes, and known issues.

The IntelProcessorDiagnosticToolHelp User Guide is available for additional tool information.
