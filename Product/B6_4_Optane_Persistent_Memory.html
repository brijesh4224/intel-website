Features and Performance

Affordable Larger Capacity

PMem enables you to affordably increase your memory capacity per socket, extract value from larger data sets and increase the utility of the server. PMem can help improve TCO via reduced SW licensing costs, power efficiencies and more.

Maximizing Your Investments

PMem is propelling infrastructure consolidation. Larger memory helps consolidate workloads, concentrating them on fewer nodes, saving on deployments and helping maximize previously underutilized processors due to memory constraints.

Driving Innovation

Intel® Optane™ PMem is an advanced memory technology with groundbreaking performance to break-through system bottlenecks that severely constrain workloads.
