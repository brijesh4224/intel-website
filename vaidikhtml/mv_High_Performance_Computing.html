How to Select Processors and Accelerators for Your HPC Environments

HPC use cases require different compute configurations based on their application’s code. Across the full range of HPC architectures and compute models, we can help you to accelerate performance, maximize scalability, and enhance results. Whether you’re looking to offload tasks to a discrete accelerator or enhance the performance of demanding workloads via built-in accelerators and instruction set architecture (ISA) improvements, our portfolio can help.

General-Purpose Processors

General-purpose processors offer a range of core counts and sizable memory bandwidth to accommodate complex distributed HPC workloads.

For the best results and to help avoid performance issues when running advanced applications in HPC environments, you should select general-purpose processors that offer built-in accelerations, optimizations, and instruction sets for demanding HPC workloads and computational tasks.

Likewise, these built-in accelerators mean that HPC application developers can rely on the architecture they already know without the need for specialized GPU or accelerator programming skills.
