How to Install Intel® Arc Control

Content Type: Install & Setup | Article ID: 000089899 | Last Reviewed: 02/03/2025

For detailed installation instructions on how to install Install Intel® Arc Control, see below.

Minimum System Requirements for Intel® Arc Control

  • 11th Generation Intel® Core processor or newer or an Intel® Arc™ Dedicated Graphics card
    • See How to Identify your Intel® Processor and How to Find the Generation of Intel® Core™ Processors
  • Windows® 10 version 2004 or higher
  • Windows 11* version 21H2 or higher
  • Intel® Graphics driver version 30.0.101.1325 or newer
    • See How to Identify Your Intel® Graphics Driver Version in Windows® 10 & Windows 11*
  • Intel® Driver & Support Assistant version 22.1.8.8 or newer
  • Internet connection
Note The Intel® Arc Control application is not supported on Linux*.

Steps to Install

  1. Check your system meets the minimum system requirements listed above.
  2. Download and install the Intel Graphics 31.0.101.3277 or newer. The installer will install the Intel Graphics driver and Intel Arc Control components. For driver installation instructions, see How to Install an Intel® Graphics Driver in Windows® 10 & Windows 11*.
  3. Reboot the system once the driver installation is complete.
